1:安装依赖
    pip install selenium -i http://pypi.douban.com/simple/
    pip install PyQt5 -i http://pypi.douban.com/simple/
    pip install scrapy -i http://pypi.douban.com/simple/

2：下载Chrome驱动：(不推荐其他浏览器，慢，不稳定)
    ①：查看自己Chrome的版本
    ②：下载对应自己版本的驱动, 替换项目中的chromedriver
        科学上网：https://sites.google.com/chromium.org/driver/

        如果打不卡上面的网址，从下面入口进入
        ⑴ https://selenium-python.readthedocs.io/installation.html
        ⑵ 找到1.5 Drivers， 点击(科学上网) Chrome：后面的地址

支持Windows，Mac，Ubuntu，CentOS
